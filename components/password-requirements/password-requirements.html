Password Requirements

  • Passwords must match 
  • Passwords must contain between 8–50 characters 
  • Passwords must contain least 1 upper-case letter 
  • Passwords must contain least 1 lower-case letter 
  • Passwords must contain least 1 number 
  • Passwords must contain least 1 of the following characters: ! @ # $ % ^ & * ( ) ' - + , . / : ; < = > ? [ ] _ ` { } | \  
  • Passwords must contain between 8–50 characters 
  • Passwords must contain least 1 upper-case letter 

When using verification methods, you can insert the class .valid or .invalid to <li class="default"> that will trigger a css change to highlight password requirements either not met or met.

If not using these validation display methods, please change the .css for ul.requirements li.default span i.la.la-circle to display: none;.
